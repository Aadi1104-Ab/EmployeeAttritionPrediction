# Employee Attrition Prediction using Machine Learning

## Objective
This project predicts whether an employee is likely to leave the company using machine learning techniques and provides actionable insights for HR.

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Models Used
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
